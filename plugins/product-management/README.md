# Product Management

Focused plugin for product and delivery workflows:

- **write-ticket** — Implementation-ready Jira-style issues
- **summarize-board** — Board or sprint summaries
- **devils-advocate** — Challenges plans and assumptions before build (Jira-aware via MCP)
- **Atlassian MCP** in `mcp.json` for Jira/Confluence when your org uses it

Plugin id: **`product-management`**. Display name: **Product Management**.
