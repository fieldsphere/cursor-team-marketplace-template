# Design

Focused plugin for UI/UX work:

- **wireframes** skill — Backend-aware wireframes and data states
- **mockup** skill — Production-oriented mockup workflow
- **component-designer** agent — Component-level design and specs
- **ui-engineer** agent — UI implementation aligned with design intent
- **Figma MCP** in `mcp.json` when your team uses Figma

Plugin id: **`design`**. Display name: **Design**.
